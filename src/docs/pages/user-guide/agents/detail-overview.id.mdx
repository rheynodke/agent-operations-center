### Compact mode

Toggle ikon di kanan header untuk **compact header** (~40px single-row). Persisted di `localStorage` `aoc.agent-detail.headerCompact`. Berguna kalau workspace kamu butuh ruang vertikal lebih.

## Edit Configuration Modal

Klik tombol **⚙ Edit Config** di header. Modal `EditConfigModal` muncul:

- **Display name** — bisa diubah. Kalau berubah, **slug ID** auto-recompute (kecuali Master, slug-nya di-protect karena banyak referensi).
- **ADLC role dropdown** — pilih dari 9 role + master-orchestrator. Locked ke "master-orchestrator" kalau `profile.isMaster=true` — gak bisa diubah.
- **Model** — pilih dari list model yang available di gateway kamu (read dari `openclaw.json` `models.list[]`).
- **Persona description** — short description.
- **Color** — palette swatch.
- **Avatar preset** — 16 preset (matching onboarding wizard).

<Callout type="danger" title="Rename = atomic cascade">
Saat display name berubah dan slug auto-recompute beda dari sebelumnya:

1. `openclaw.json` — agent entry key di-rename
2. `channels.telegram.accounts` / `channels.whatsapp.accounts` — key matching old ID di-rename
3. Semua binding `agentId` references di-update
4. Filesystem: `<userHome>/agents/{id}/` + workspace dir di-rename
5. SQLite `db.renameAgentProfile(oldId, newId)` di-call

Frontend navigate ke `/agents/<newId>` setelah sukses. Atomic — kalau salah satu step fail, di-rollback.

**Master Agent slug `main` di-protect** — tidak bisa di-rename karena banyak komponen lain reference literal `'main'` (legacy).
</Callout>
